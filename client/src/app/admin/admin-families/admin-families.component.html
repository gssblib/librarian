Downloading the Families Report

This step requires full reporting access to Sycamore.

  1. Go to Sycamore Education.
  2. Log in using your admin account.
  3. Go to Administrative > Back Office Mgr > Reports
  4. Select Dynamic Student / Family Report
  5. Select Properties
  6. Select the following fields:
    • Family Information : Family Code (Sycamore ID)
    • Family Information : Family Name
    • Family Information : Cell Phone Number(s)
    • Family Information : Email Address(es)
    • Student Information : Last Name
    • Student Information : First Name
    • Student Information : Email Address
  7. Click Generate CSV
